# Groundswatch
**Grounds Watch is a platform that connects hosts and event organizers to different venues. It features information about the venues like images, reviews, capacity, and prices.**

groundswatch.onrender.com
=========================


Technologies used
-----------------
1. *Node v12.22.12*
2. *mongoose@5.13.16*
3. *Bootsrap 5.0.2*
3. *Express*
4. *MongoDB Atlas*
5. *Render*
6. *Cloudinary*
